# Figure 2 通信图结构图

## 图形内容

本图展示四种OLE协议的有向加权图结构，分为子图a至d。每个子图包含两个圆形节点 $v_1$ 和 $v_2$ 表示协议参与方，节点间的弧线表示消息传输边，弧线粗细与边权 $w(e)$ 成正比，弧线方向表示消息流向。子图a为Paillier-OLE，子图b为MASCOT-OLE，子图c为SPDZ-OLE，子图d为Silent-OLE。

从图中可以观察到各协议的边数量存在差异，MASCOT-OLE的边数为10条，SPDZ-OLE为8条，而Paillier-OLE和Silent-OLE均为4条。边的粗细分布也呈现不同特征，Paillier-OLE的边较粗反映同态加密密文的通信开销，Silent-OLE的边较细体现其亚线性通信特性。

## 理论支撑

该图为论文的图论建模方法提供可视化呈现。图论模型将协议形式化为有向加权图 $G = (V, E, w)$，其中 $V = \{v_1, v_2\}$ 为顶点集，$E$ 为边集，$w: E \to \mathbb{Z}^+$ 为权函数。

图中边的数量对应 $|E|$，边权总和对应 $W(G) = \sum_{e \in E} w(e)$。边粗细的不均匀程度与权重集中度 $G_w$ 相关，该指标采用归一化基尼系数计算。两个方向的边权之比对应方向平衡度 $\beta$，图中可见部分协议的双向边粗细接近，表明 $\beta$ 接近1.0。
