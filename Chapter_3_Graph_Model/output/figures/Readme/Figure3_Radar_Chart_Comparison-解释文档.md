# Figure 3 雷达图比较

## 图形内容

本图采用雷达图形式对四种代表性协议进行多维度比较，包括P1、P4、P8和P0。雷达图的五个轴分别对应总边权 $W$、边数量 $|E|$、时序熵 $H$、权重集中度 $G_w$ 和方向平衡度 $\beta$，各协议的指标值经归一化处理后映射到0至1范围内，每个协议对应一个封闭多边形。

从图中可见，P1在 $W$ 轴上数值较高，反映同态加密协议的通信开销特点。P4在 $|E|$ 和 $H$ 轴上数值较高，对应OT扩展协议多轮多消息的结构。P8整体轮廓较小，体现亚线性通信协议的效率特性。P0作为基线协议，在 $\beta$ 轴上接近边界，对应其对称通信结构。

## 理论支撑

该图支持论文关于不同密码学基础类别协议特征差异的分析。归一化采用最小最大方法

$$m_{norm} = \frac{m - m_{min}}{m_{max} - m_{min}}$$

使不同量纲的指标可在同一坐标系比较。多边形轮廓的形状差异反映了各协议在通信复杂度、时序结构和对称性等方面的综合特征，为协议选择提供参考依据。
