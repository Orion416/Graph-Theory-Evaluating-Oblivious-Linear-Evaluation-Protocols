# Figure 4 协议分布散点图

## 图形内容

本图采用散点图展示九种OLE协议在边数量与总边权二维空间中的分布。横轴表示总边权 $W$ 以比特为单位，纵轴表示边数量 $|E|$，每个散点代表一个协议并标注协议编号。散点的形状和颜色区分密码学基础类别，图例标注HE、OT、Algebraic、Sublinear和Baseline五类。图中虚线为等积曲线。

从图中可见，P1位于横轴右侧区域，反映同态加密协议的高通信量特点。P4、P5、P7分布在纵轴较高位置，对应OT扩展协议的多边结构。P3、P6属于代数类，P8属于亚线性类，P0作为基线位于左下角。同类协议在空间中呈现聚集分布的趋势。

## 理论支撑

该图支持论文关于密码学基础对协议通信特征影响的分析。等积曲线满足 $|E| \times W = c$ 的关系，位于同一曲线上的协议具有相同的边权乘积但分配不同。这种可视化方式有助于理解不同协议在消息数量与消息大小之间的权衡关系，为特定应用场景下的协议选择提供参考。
