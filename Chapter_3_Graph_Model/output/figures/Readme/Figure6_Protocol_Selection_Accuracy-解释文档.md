# Figure 6 协议选择准确性图

## 图形内容

本图采用柱状图展示基于图论度量的协议选择方法与专家判断之间的一致性得分。横轴表示十个验证场景S1至S10，纵轴表示一致性得分范围0至1，柱高反映两种方法在对应场景下的吻合程度。图中水平虚线表示0.8的阈值参考线，柱的颜色根据得分高低区分。左上角标注平均得分Avg和完全一致场景数Full。

从图中可见，S2至S5以及S7至S10的柱高达到1.0，表明这些场景下两种方法选择了相同的协议。S1和S6的柱高较低，S1为0.0表示选择不一致，S6为0.25表示部分一致。统计显示平均一致性得分为0.82，完全一致场景数为8/10。

## 理论支撑

该图用于评估基于图论模型的协议选择算法有效性。十个验证场景涵盖不同约束条件组合，包括安全性需求、延迟约束和带宽限制等，场景约束阈值基于协议度量的统计百分位数设定。一致性得分采用分级评分机制，完全相同选择得1分，度量相近的不同选择得0.5分，其他情况得0分。平均一致性得分0.82表明图论方法的选择结果与专家判断具有较好的吻合度。
