# Figure 5 信息保持度比较图

## 图形内容

本图采用热力矩阵形式比较四种协议描述方法在保持通信特征方面的能力。矩阵列对应四种描述方法，分别为NL自然语言、PC伪代码、AN渐近记号和GM图论模型。矩阵行对应四种通信特征维度，分别为Dir.消息方向、Size消息大小、Temp.时序和Round轮次。单元格颜色深浅表示保持程度，颜色越深表示保持能力越强，右侧色标范围为0至1。

从图中可见，NL列各单元格颜色均较浅，表明自然语言在保持协议结构特征方面能力有限。PC列在Dir.、Temp.和Round维度呈现较深颜色，但Size维度较浅。AN列在各维度的颜色均较浅。GM列在全部四个维度均呈现深色，表明图论模型能够保持消息方向、大小、时序和轮次等多方面特征。

## 理论支撑

该图支持论文关于不同描述方法表达能力的比较分析。保持度计算基于各方法的形式化能力，采用布尔特性聚合的方式得出，如方向保持度由方向表示能力和结构表示能力加权组合计算。图论模型作为形式化工具，其边集 $E$、权函数 $w$ 和时序序列 $T$ 可分别承载方向、大小和时序信息，因此在各维度均具有较高的保持能力。
