# Figure 7 模型与基准测试对比图

## 图形内容

本图展示图论模型预测值与基准测试实测值的对比分析，分为左右两个子图。左侧子图a展示通信量对比，横轴为协议编号，纵轴为通信量以字节为单位，每个协议对应两个柱分别表示模型预测值和实测值，子图内标注Spearman相关系数 $\rho$。右侧子图b展示计算开销对比，模型值单位为AES等效操作数，实测值单位为毫秒，同样标注 $\rho$ 值。

从图中可见，各协议的模型预测通信量与实测值呈现一致的排序关系，$\rho = 1.0$ 表明排名完全一致。实测值略高于模型值，平均误差约11%，这反映了实际实现中序列化和协议握手带来的额外开销。计算开销的 $\rho$ 值同样为1.0，表明模型对计算复杂度排名的预测与实测结果一致。

## 理论支撑

该图为论文的外部验证部分提供支撑。图论模型中边权 $w(e)$ 的预测值与实测通信量的高相关性验证了通信建模的有效性。计算权重采用AES等效操作数作为标准化单位，其与实测时间的排名一致性验证了计算建模方法的可行性。通过同时展示通信和计算两个维度，该图支持论文关于协议瓶颈应同时考虑带宽和计算开销的分析框架。基准测试数据由物理模型模拟生成，当真实库测量数据可用时可进行替换验证。
