# Figure 1 时序通信模式图

## 图形内容

本图展示四种代表性OLE协议的时序通信模式，分为子图a至d。每个子图中，两条水平线表示参与方 $v_1$ 和 $v_2$，垂直箭头表示消息传输，箭头方向指示消息流向，箭头粗细与消息大小 $w(e)$ 成正比。横轴标注R1、R2等表示通信轮次。

子图a展示Paillier-OLE协议的3轮通信结构，子图b展示MASCOT-OLE协议的5轮结构，子图c展示SPDZ-OLE协议的4轮结构，子图d展示FSS-OLE协议的3轮结构。不同协议在轮次数量、每轮消息数量以及消息大小分布上呈现出各自的特征。

## 理论支撑

该图支持论文关于时序熵度量 $H_t$ 的论述。时序熵定义为

$$H_t = -\sum_{r=1}^{R} p_r \log_2 p_r$$

其中 $p_r = W_r / W_{total}$ 表示第 $r$ 轮通信量占总通信量的比例。当通信量在各轮之间分布均匀时，时序熵较高；当通信量集中于少数轮次时，时序熵较低。

图中可观察到MASCOT-OLE协议的通信分布在5轮中相对均匀，对应较高的时序熵值2.24；而部分2轮协议的通信分布不均，对应较低的时序熵值。箭头方向的对称性反映了边方向平衡度 $\beta = w(v_1 \to v_2) / w(v_2 \to v_1)$ 的特性，DDH-OLE等协议呈现对称结构，平衡度接近1.0。
